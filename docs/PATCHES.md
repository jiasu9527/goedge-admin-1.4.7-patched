# 补丁内容说明

这个仓库里的补丁版是从已部署机器当前运行目录回收整理出来的，目标是让其他同版本机器可以直接覆盖升级。

## 已整理进发布包的内容

- `bin/edge-admin`
- `web/views/@default`
- `configs/plus.cache.json`

## 这次补丁涉及的方向

### 1. 商业授权状态固定为可用

- 补丁版带入了 `configs/plus.cache.json`
- 用于让商业授权相关功能保持可用

### 2. 商业版菜单与版本显示修正

- 之前出现过设置里商业授权入口缺失、版本显示异常
- 当前回收目录已经包含修正后的模板与运行二进制

### 3. 证书页面可用性调整

- 已做证书页交互修正
- 包含手动续期入口相关调整
- 包含批量选择/批量删除相关页面修正
- 包含页面拥挤显示问题的模板修复

### 4. 首页仪表盘无数据问题修复

已在模板层增加兼容处理：

- 节点访问排行：从 `/clusters/cluster/boards` 聚合回退
- 国家/地区排行：从 `/servers/server/stat/regions?serverId=...` 聚合回退

这样即使原始 `/dashboard/boards` 接口返回空，也能在前端正常出图。

## 未包含进仓库的内容

为了避免泄露线上配置，以下内容没有进入发布包：

- `configs/server.yaml`
- `configs/api_admin.yaml`
- `configs/api_db.yaml`
- `edge-api/**`
- `data/**`
- `logs/**`

## 已知边界

- 这是覆盖升级包，不是全新初始化安装包
- 适用前提是目标机本身已经安装 `GoEdge Admin 1.4.7`
- 这个仓库当前只整理了 `amd64` 版本
